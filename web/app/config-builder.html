💡 LED Channels Required ›

Each channel independently dims one LED circuit. Toggle a channel Active to configure it, then set a default brightness and optional schedule windows for time-based control.

🔌 Relays Optional ›

Relay outputs for switching non-dimmable loads — fans, UV lamps, power rails, etc. Set time windows for scheduled on/off.

👁 Motion Sensors (PIR) Optional ›

Up to 4 PIR motion sensors. When motion is detected the chosen action overrides the schedule. When the area goes quiet the vacancy action runs — usually reverting back to the schedule.

🎬 Scenes Optional ›

A scene is a named lighting snapshot — e.g. motion_bright, evening, meditation. Scenes are partial: only the outputs you tick are changed; everything else keeps its current state. Triggered by motion sensors, the web dashboard, or LoRa remote commands.

☀️ Ambient Light Sensor (LDR) Optional ›

The light sensor measures ambient brightness and automatically caps LED output. Useful outdoors — if it's still bright outside, LEDs won't blare at full power. This is a cap only: it reduces brightness, never increases it.
