Lokki Configuration Tools

Static tools for configuring and setting up Lokki LED lighting controllers. Runs entirely in your browser — nothing is uploaded.

⚙️
Config Builder
Build a config.json for any Lokki unit — coordinator or leaf. Configure LED schedules, PIR motion responses, relay control, scenes, and LoRa network settings. Includes an integrated sun times generator for solar-based schedules.

Setup Workflow

Before you start — you'll need:
  • MicroPython UF2 for Pico 2 W (coordinator) and Pico 2 (each leaf) — from micropython.org
  • A file transfer tool: Thonny IDE (beginner-friendly) or mpremote via pip install mpremote
  • This repo's firmware/micropython/src/ and web/app/ folders cloned locally
1
Build config
Open Config Builder, choose role Coordinator or Leaf, fill in your settings, and download config.json. If any LED schedule uses sunrise or sunset, the Sun Times section appears automatically — set your location and download sun_times.json too. 🌐 Coordinator: enter your WiFi credentials and the unit IDs of your leaf units. 🍃 Leaf: set a unique unit ID (1–8). No WiFi credentials needed.
2
Flash MicroPython
Hold BOOTSEL while plugging in USB — the board mounts as a drive. Drag the UF2 onto it. The board reboots into MicroPython. 🌐 Coordinator: use the Pico 2 W UF2. 🍃 Leaf: use the Pico 2 UF2 (no W).
3
Upload firmware
Using Thonny or mpremote, copy the entire src/ folder to the board root. 🌐 Coordinator only: also copy web/app/ to a folder named /www on the board — this serves the web dashboard over WiFi.
4
Upload config
Copy config.json (and sun_times.json if generated) to the board's filesystem root — same level as the src/ folder.
5
Power on
Unplug and reconnect power. 🌐 Coordinator connects to WiFi and starts the web dashboard. Find it at its router-assigned IP, or try lokki-<unitname>.local. 🍃 Leaf starts up and listens for the coordinator over LoRa. No WiFi needed — time and commands sync automatically.
🔄
Coordinator: updating config without USB
Once the coordinator is running, open its web dashboard in a browser and navigate to Config Builder. Make your changes and click Save to device — the config uploads directly over WiFi. Reboot the coordinator (power cycle or reboot button in the dashboard) to apply the new config. No USB cable needed after the initial setup.
✅
Verify everything is working
Open the coordinator's web dashboard in a browser. The Fleet table should show all leaf units as Online within a minute or two of powering them on. If a leaf shows Offline, check that its unit ID matches what the coordinator has in its Leaf Unit IDs config, and that all units are on the same LoRa channel and frequency.
