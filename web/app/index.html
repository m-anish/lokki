Lokki Configuration Tools

Static tools for configuring and setting up Lokki LED lighting controllers. Runs entirely in your browser — nothing is uploaded.

⚙️
Config Builder
Build a config.json for any Lokki unit — coordinator or leaf. Configure LED schedules, PIR motion responses, relay control, scenes, and LoRa network settings.
Open tool →
🌅
Sun Times Builder
Generate a sun_times.json lookup table for your location. Required when you use sunrise or sunset keywords in your LED schedules.
Open tool →

Setup Workflow

1
Build config
Use the Config Builder to create a config.json for each unit.
2
Sun times (if needed)
If your schedule uses sunrise/sunset, generate a sun_times.json for your location.
3
Flash firmware
Copy MicroPython firmware to each Pico 2. Upload the src/ folder.
4
Upload configs
Copy config.json (and sun_times.json) to each unit's filesystem root.
5
Power on
Units boot, the coordinator connects to WiFi, leaves sync via LoRa. Done.
